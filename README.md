# 🛠️ RGExtractor

**RGExtractor** is a custom utility made to automate the building process in **BRM5's RGE** made for Monkey Business

The project works as a unified system and consists of two parts: a Roblox Studio plugin and a local macro script running on your PC.

---

> **Project Status: Early Release**
> This is a very raw and experimental version of the tool (`v0.1.3`). The code is still fresh out of the oven, so things might break. I am actively working on this project, and huge feature updates are dropping soon. Feel free to test it out.

## ⚙️ How It Works

The tool automates the building process through a streamlined two-step workflow:
* **The Plugin:** Extracts technical data about the selected parts in Roblox Studio, compresses it, and formats it for the script.
* **The Macro:** Reads this compressed data from the local file. It then automatically duplicates pre-made in-game parts of all types (using custom names), positions them precisely in the game world, and applies the extracted parameters.

## 📖 Tutorial / How to Use

### Step 1. Prepare the files
1. Extract the downloaded `macro-v0.1.3.zip` archive into any folder
2. Make sure there is a file named `data.txt` *in the same directory*. If it's missing, simply create an empty text file and name it `data.txt`.

### Step 2. Launch the system
3. Open Roblox Studio, select a **grouped object** (Model), and use the plugin to extract data. For more detailed instructions, the plugin provides its own README script.
4. Paste extracted data into your `data.txt` file.
5. Run `rgextractor_macro.exe`

**Crucial:** Always run the extracted `.exe` file **as Administrator**.
### Step 3. Control the macro in-game
6. 🧱 **Crucial Prep:** Make sure you have manually placed at least one part of **every single type** (Block, Wedge, Cylinder, etc.) into your RGE workspace and gave it a custom name. The macro functions strictly through **duplication**. If a specific part type isn't already on the map for the macro to copy, it won't be able to build it.
7. Just follow the instructions in the macro.

---

## Notes for use

* Do **NOT** move your mouse or press any keys on your keyboard while the macro is actively building in-game.
* The plugin only works with standard, default Roblox Studio parts (`BasePart`, `Part`, `Wedge`, etc.). It **will not recognize** or export custom parts or objects generated by third-party plugins.
* Windows Defender or other security software might flag the `.exe` file because Python scripts is capturing keystrokes.
